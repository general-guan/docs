# TCP

## 计算机网络

| 计算机网络 | 俗称   |                                                                                                                                                        |
| ---------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| LAN        | 局域网 | Local Area Network，读作 “lang”，指的是将位于大学、研究所或者企业内部等较狭小空间内的设备连接起来的网络。<br />常用细铜线连接。                        |
| WAN        | 广域网 | Wide Area Network，读作 “wang”，指的是将类似于分公司之间的地理位置上互相远离的设备连接起来的较大规模的网络。<br />常用光纤以及公用网（电话）线路连接。 |
| 因特网     |        | 将多个 LAN 及 WAN 连接起来的全球规模的网络。                                                                                                               |

## TCP

TCP/IP 共分为 5 层

越靠上越接近用户，越靠下越接近设备

| 层 layer   | 协议                                  |                                                                                                                  |
| ---------- | ------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| 应用层     | HTTP SMTP POP3 FTP TELNET NNTP RCP... | 让通信符合应用程序<br />各个应用程序有各种各样的协议支持                                                         |
| 传输层     | TCP UDP                               | 确保发出的数据能够切实到达收件人的应用程序处<br />分割/组装数据                                                  |
| 网络层     | IP                                    | 负责将数据送达收件人的计算机<br />不关心送达的数据是否损坏，或者收件人是否收到                                   |
| 数据链路层 | Ethernet FDDI ATM PPP PPPoE...        | 确保能够在直接连接于网络上的设备之间传递。为了完全清除网络层与物理层的不同，准备有各种各样的协议<br />二进制数列 |
| 物理层     |                                       | 负责将数据转换为信号，将信号转换为数据。转换方法因通信媒体不同而不同，所以没有特定的协议                         |

一般情况下，TCP/IP 指的是上述 5 层的整体。但有时也指 TCP 和 IP 这两个单独的协议。为了避免混淆，有时将 5 层协议叫做 TCP/IP 协议族。
